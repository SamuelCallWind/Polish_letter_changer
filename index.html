Welcome to the Polish letter changer page

This small program is designed to change the incorrectly encoded Polish letters in ISO 8859-2 and ISO 8859-1, ensuring their proper display in a text file. It generates a new file that rectifies these encoding errors

Download
Choose a file (.srt / .txt file)
Once the file is selected, click on "Upload file"
Click on "Download" to get your file changed
